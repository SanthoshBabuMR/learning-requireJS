Exploring requireJS Library


**Dependencies**
* grunt-contrib-connect

**Launching server from grunt**
* Web server
```bash
grunt
```
* "Cross-Orgin Resource Sharing" enabled server
```bash
grunt corsServer
```
* Non-"Cross-Orgin Resource Sharing" server
```bash
grunt nonCorsServer
```
* run a server using ip address
```bash
grunt serverIp
```
