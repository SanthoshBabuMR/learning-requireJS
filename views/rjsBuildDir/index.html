rjs Build Dir

  1. GoTo /buildScripts/requireJSBuildDir.js
  2. Configure your build options for r.js
  3. run cmd > node r.js -o requireJSBuildDir.js
  4. The above command will scan the "modules" property looking at the AMD modules to optimize and move the optimized files to the directory specified in "dir" property
  5. open /buildDir/ directory to view the optimized files
