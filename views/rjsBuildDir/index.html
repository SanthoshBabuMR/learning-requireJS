rjs Build Dir

  1. GoTo /buildScripts/requireJSBuildDir.js
  2. Configure your build options for r.js
  3. run cmd > node buildScripts/r.js -o buildScripts/<builtScriptFile>.js from your root directory
  4. The above command will scan the "modules" property looking at the AMD modules to optimize and move the optimized files to the directory specified in "dir" property
  5. open /buildDir/ directory to view the optimized files

cmd(s) for buildScript ( to be run from root directory )

directory build using normalizeDirDefines config option
node buildScripts/r.js -o buildScripts/requireJSBuildDirUsingNormalizeDirDefines.js
directory build using skipDirOptimize config option
node buildScripts/r.js -o buildScripts/requireJSBuildDirUsingSkipDirOptimize.js
simple directory build
node buildScripts/r.js -o buildScripts/requireJSBuildDir.js
