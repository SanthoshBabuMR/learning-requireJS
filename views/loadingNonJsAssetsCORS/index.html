For CORS to work as expected

  • requireJS module needs to be ensured that it returns "true" for "useXhr" function. Checkout /public/scripts/config/configLoadingNonJsAssetsCORS.js
  • The server being requested for assets/resource must have the corresponding headers ( Access-Control-Allow-Origin, Access-Control-Allow-Methods ) set to allow CORS
Ensure all servers ( grunt, grunt corsServer, grunt nonCorsServer ) are started
