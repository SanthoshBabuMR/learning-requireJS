Exploring enforceDefine

enforeDefine: true

  • implies all AMD modules must use only define() and not require()
  • The above implies that, if requirejs.config() was included from a separate file, then it needs to wrapped in define()

Notes

If you are seeing this, it means the modules have not been loaded, please do the following

  • For the AMD modules to execute, goto /public/scripts/config/configExploringEnforceDefine.js and wrap the configuration in a define().
  • goto public/scripts/config/configExploringEnforceDefine.js and update require() to define()
