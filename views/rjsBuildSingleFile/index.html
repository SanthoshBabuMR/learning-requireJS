rjs Build Single File

  1. GoTo /buildScripts/requireJSBuildSingleFile.js
  2. Configure your build options for r.js
  3. run cmd > node buildScripts/r.js -o buildScripts/<builtScriptFile>.js from your root directory
  4. With the above command, ensure you provide the entry js file into "name" property and the optimized file name to be provided in "out" property. This option is configured for single file optimization.
  5. open /buildSingleFile/ directory to view the optimized file(s)

cmd(s) for buildScript ( to be run from root directory )

single file build using inlineText config option
node buildScripts/r.js -o buildScripts/requireJSBuildSingleFileInlineText.js
single file build using cssPrefix config option
node buildScripts/r.js -o buildScripts/requireJSBuildSingleFilecssPrefix.js
single file build using optimizeCss config option
node buildScripts/r.js -o buildScripts/requireJSBuildSingleFileOptimizeCss.js
single file build using uglify config option
node buildScripts/r.js -o buildScripts/requireJSBuildSingleFileUsingUglify.js
See https://github.com/mishoo/UglifyJS for the possible values.
single file build using uglify2 config option
node buildScripts/r.js -o buildScripts/requireJSBuildSingleFileUsingUglify2.js
For possible values see:
  • http://lisperator.net/uglifyjs/codegen
  • http://lisperator.net/uglifyjs/compress

single file build using wrapShim config option
